# loveosapk

Android-приложение для пар: совместные активности, трекеры, заметки, wishlists, задачи, игры и синхронизация данных между партнерами.

## Возможности

- главный экран с таймером отношений и идеями для пары;
- женский календарь и partner sync;
- wishlist, задачи, накопления и заметки;
- встроенные мини-игры;
- локальное хранение через Room и настройки через DataStore;
- синхронизация через Firebase.

## Стек

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Room + KSP
- Firebase Realtime Database
- Firebase Storage
- Firebase Auth
- DataStore
- Coil 3
- Media3

## Запуск

```bash
./gradlew assembleDebug
```

Параметры Android:

- `applicationId`: `com.example.loveosapk`
- `minSdk`: 24
- `targetSdk`: 36
- `compileSdk`: 36
